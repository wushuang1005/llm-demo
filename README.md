# Project Overview
This project is a demonstration of large language models (LLMs).

## Description
The LLM Demo showcases various functionalities and applications of large language models, providing insights into their capabilities and potential uses.

## Features
- Model demonstration
- Interactive queries
- Educational examples

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/wushuang1005/llm-demo.git
   cd llm-demo
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the demo, execute the following command:
```bash
python demo.py
```
